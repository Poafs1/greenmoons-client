# Greenmoons interview assignment

## Requirement

1. `.env`

## Installation

```sh
yarn install
```

## Run

### Locally

```sh
yarn dev
```

### Docker

```sh
docker-compose -f docker-compose.dev.yaml up --build
```

## Unit Test

```sh
yarn test
```

## Storybook (Just a small example)

yarn storybook
